# 🎬 Netflix Data Analysis using Python

## 📌 Project Overview

This project analyzes the Netflix Movies and TV Shows dataset using Python. The objective is to perform Exploratory Data Analysis (EDA) to uncover trends, patterns, and insights about Netflix's content library. The analysis includes data cleaning, visualization, and interpretation of key business insights.

---

## 🎯 Objectives

- Analyze Netflix's content library.
- Compare Movies and TV Shows.
- Identify the most common genres.
- Find the countries producing the most Netflix content.
- Analyze content added over the years.
- Discover the most frequent content ratings.
- Visualize insights using charts.

---

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Jupyter Notebook

## 📂 Dataset

The dataset contains information about Netflix titles, including:

- Show ID
- Title
- Type (Movie / TV Show)
- Director
- Cast
- Country
- Date Added
- Release Year
- Rating
- Duration
- Genre
- Description

---

## 🔑 Key Insights

- Movies make up a larger portion of Netflix's content than TV Shows.
- The United States contributes the highest number of titles.
- Drama and International Movies are among the most common genres.
- Netflix significantly expanded its content library after 2015.
- TV-MA is one of the most frequently assigned content ratings.
